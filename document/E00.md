# E00 초기설정

- 1. DB 유저생성

```
	CREATE USER 'example'@'%' IDENTIFIED BY 'example';
	GRANT ALL PRIVILEGES ON example.* TO 'example'@'%';
```

- 2. TABLE 생성

```
	create table t_board
	(
	    BOARD_SEQ int auto_increment
	        primary key,
	    TITLE     varchar(10) null,
	    CONTENTS  varchar(30) null,
	    REG_DATE  datetime    null
	)
	    charset = utf8mb3;
```
